# Injective Protocol Golang SDK 🌟

---

## 📚 Getting Started

Clone the repository locally and install needed dependencies

```bash
$ git clone git@github.com:equityLab/sdk-go.git
$ cd sdk-go
$ go install ./...
```

---

## Publishing Tagged Release

```bash
$ git add .
$ git commit -m "bugfix"
$ git tag -a v1.1.1
$ git push origin master --tags
```
